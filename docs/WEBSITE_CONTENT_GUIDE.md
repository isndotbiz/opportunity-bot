## Content Tone & Style Guide

### Voice
- **Professional but approachable** - Investor-ready without being overly formal
- **Technical but clear** - Show expertise without jargon overload
- **Results-focused** - Emphasize measurable outcomes
- **Innovation-driven** - Highlight cutting-edge approaches

### Writing Guidelines
- Use active voice
- Lead with benefits, follow with features
- Include specific metrics (not "many" but "12.5M+")
- Show, don't just tell (code samples, diagrams)
- Balance technical depth with accessibility

### Formatting
- Short paragraphs (2-3 sentences max)
- Bullet points for lists
- Bold for emphasis
- Code blocks for technical details
- Icons for visual interest
- Tables for comparisons
